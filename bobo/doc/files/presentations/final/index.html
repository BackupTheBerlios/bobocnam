Sujet de notre présentation

  • Publié sous licence libre : GPL

  • Notre client : AirCnam, représenté par M. Berne

Besoins du client

  • Système de réservation pour des vols de différentes compagnies
  • Système distribué : RMI/CORBA
  • Accès public : consultation des vols, enregistrement des réservations
  • Accès privilégié : mise à jour de la base de données
  • Sécurité : gestion des profils utilisateur

Notre solution

  • Projet développé en Java
  • Développement d'un logiciel libre basé exclusivement sur des composants open source :
    • Spring framework
    • Hibernate
    • XDoclet
    • Maven
  • Modulaire

Architecture

Différents composants :

  • Modèle de données : bobo-model
  • Implémentation du modèle avec Hibernate : bobo-model-hibernate
  • Serveur RMI : bobo-rmiserver
  • Interface web : bobo-web
