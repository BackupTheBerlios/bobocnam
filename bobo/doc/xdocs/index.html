Bienvenue !

Le projet Bobo est un logiciel développé dans le cadre d'un sujet d'étude pour le module Conduite de projet du CNAM d'Aix-en-Provence/Toulon pour la session 2004/2005. Le but de ce projet est de concevoir un système de réservation en ligne en Java, dans lequel la technologie CORBA est utilisée pour l'accès aux données.

Pour les plus impatients d'entre-vous, nous avons prévu une démo, pour apprécier le projet sur votre ordinateur en moins de 5 minutes !

Deux présentations sont disponibles :

  • Intégration, validation, maintenance
  • Présentation finale
