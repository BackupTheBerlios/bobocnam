Démo

La démo que nous vous proposons se décompose en deux parties.

  1. Bobo RmiServer : serveur RMI en connexion avec une base de données ;
  2. Bobo WebRunner : serveur web embarquant l'application web se connectant au serveur RMI pour l'accès aux données.

Pré-requis

Ces deux applications sont disponibles sous la forme d'applications JNLP. Java WebStart permet de déployer des applications sur un poste client en toute simplicité ! Vous devez installer Java (version 1.4 ou plus) avant de pouvoir lancer RmiServer ou WebRunner. Si votre installation est correcte, les applications se lanceront en utilisant le programme javaws présent dans le dossier où Java est installé.

Lancement

Le serveur RMI doit être lancé avant le serveur web pour que ce dernier puisse se connecter à une source de données.
